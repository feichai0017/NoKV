# Code of Conduct

NoKV follows the spirit of the [CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md).

This repository is a technical collaboration space for storage, databases, and distributed systems work. Treat it that way.

## Expected Behavior

- Be respectful in technical disagreement.
- Prefer evidence, code, benchmarks, and clear reasoning over rhetoric.
- Assume good faith, especially during review and design discussion.
- Keep feedback actionable and scoped to the work under discussion.
- Help maintain a welcoming environment for contributors with different backgrounds and experience levels.

## Unacceptable Behavior

- Harassment, intimidation, threats, or personal attacks.
- Discriminatory or exclusionary language or behavior.
- Deliberate disruption of discussions, reviews, or community spaces.
- Publishing private information without permission.
- Repeated bad-faith behavior after maintainers have asked for correction.

## Scope

This Code of Conduct applies to:

- GitHub issues, pull requests, discussions, and code review
- project documentation and examples
- community spaces directly tied to this repository

## Enforcement

Project maintainers may remove content, lock threads, reject participation, or ban contributors when behavior violates this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the repository owner privately through GitHub:

- [@feichai0017](https://github.com/feichai0017)

Please avoid filing a public issue for personal conduct reports unless there is no reasonable private path available.
